# TalkGenie- OpenAI-based Chatbot Android Application

# Tech Stack Used:
1. Android Studio
2. Kotlin and XML
3. OpenAi Api
4. Kotlin Coroutine
5. Glide
6. RecyclerView with Two different layouts
7. Postman (Used For Testing API)
